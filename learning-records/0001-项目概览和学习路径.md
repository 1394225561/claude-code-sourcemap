# Claude Code 源码项目概览

完成了对 Claude Code 源码的全面分析，建立了完整的学习框架。

## 关键发现

1. **项目规模**: 源码约 200+ 文件，核心代码 50 万行以上
2. **架构特点**: 分层清晰，插件化设计，事件驱动
3. **技术栈**: TypeScript + React (Ink) + Anthropic SDK
4. **核心模块**: 入口层、UI 层、查询引擎、工具系统、服务层

## 学习路径

制定了 6 个阶段的学习计划：
1. 入门基础（1-2 周）
2. 核心引擎（2-3 周）
3. 工具系统（2-3 周）
4. 服务层（2-3 周）
5. UI 和交互（1-2 周）
6. 高级主题（持续学习）
