# Mission: Claude Code 源码架构师

## Why
我想深入理解 Claude Code 这个 AI 编程助手的内部工作原理，掌握其架构设计、核心算法和工程实践。通过学习这个真实世界的大型 TypeScript 项目，提升自己的系统设计能力和 AI 应用开发技能，为将来开发类似的 AI 工具打下坚实基础。

## Success looks like
- 能够清晰解释 Claude Code 的整体架构和模块职责
- 理解 Tool 系统的设计模式和扩展机制
- 掌握 Query Engine 的对话循环和上下文管理
- 能够独立阅读和修改源码中的任意模块
- 能够基于学到的设计模式开发自己的 AI 工具

## Constraints
- 源码版本: @anthropic-ai/claude-code@2.1.88
- 学习语言: 中文为主，代码术语保留英文
- 学习方式: 渐进式，从整体到细节

## Out of scope
- Claude 模型本身的训练和推理细节
- Anthropic 公司的商业策略和产品路线图
- 其他 AI 编程工具的比较分析
