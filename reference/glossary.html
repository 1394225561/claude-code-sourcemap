Claude Code 术语表

源码中常见术语和概念的定义

架构术语

Entrypoint 架构
应用程序的入口点，负责启动和初始化。Claude Code 有多个入口点用于不同场景（CLI、MCP 服务器、SDK 等）。
entrypoints/cli.tsx - CLI 模式入口
相关: Main, REPL
Main 架构
主入口文件 (main.tsx)，包含了 Claude Code 的核心初始化逻辑、命令行参数解析、配置加载等。这是最大的文件（~800KB）。
main.tsx: runClaudeCode() 函数
相关: Entrypoint, REPL
REPL 架构
Read-Eval-Print Loop，交互式命令行循环。Claude Code 的主交互模式，持续读取用户输入、处理、输出结果。
replLauncher.tsx: launchRepl()
相关: Query Engine
Query Engine 架构
查询引擎，负责管理与 Claude API 的对话循环。处理消息发送、响应流、工具调用、错误重试等核心逻辑。
query.ts: processQuery() 函数
相关: Tool, Message

工具系统术语

Tool 工具
工具是 Claude 可以调用的功能单元。每个工具定义了输入 schema、执行逻辑和输出格式。工具系统是 Claude Code 的核心扩展点。
Tool.ts: Tool 接口定义
相关: Tool Use, Tool Result
Tool Use 工具
工具调用，指 AI 模型决定使用某个工具并传入参数的过程。在 API 响应中以 tool_use block 形式出现。
{ type: 'tool_use', name: 'Bash', input: { command: 'ls' } }
相关: Tool, Tool Result
Tool Result 工具
工具执行结果，包含工具执行的输出。需要发送回 API 以继续对话。
{ type: 'tool_result', tool_use_id: '...', content: '...' }
相关: Tool Use
Tool Permission 工具
工具权限系统，控制哪些工具可以被调用，以及在什么条件下可以执行。支持多种权限模式（自动、手动、计划模式等）。
utils/permissions/PermissionMode.ts
相关: Permission Mode
Agent Tool 工具
子代理工具，允许创建独立的 AI 代理来处理复杂任务。支持并行执行、工作隔离等高级特性。
tools/AgentTool/AgentTool.ts
相关: Subagent, Task

服务层术语

MCP 服务
Model Context Protocol，模型上下文协议。一种标准协议，用于连接 AI 模型与外部工具和数据源。Claude Code 通过 MCP 支持扩展工具。
services/mcp/client.ts
相关: MCP Server, MCP Tool
Compact 服务
上下文压缩，当对话历史超过上下文窗口限制时，自动或手动压缩历史消息以节省 token。
services/compact/autoCompact.ts
相关: Context Window
OAuth 服务
Open Authorization，开放授权协议。Claude Code 使用 OAuth 进行用户认证和 API 访问授权。
services/oauth/oauth.ts
相关: Authentication
Plugin 服务
插件系统，允许扩展 Claude Code 的功能。插件可以提供新的工具、命令、技能等。
services/plugins/pluginLoader.ts
相关: Skill
Analytics 服务
遥测和分析系统，收集使用数据以改进产品。支持多种分析后端（GrowthBook 等）。
services/analytics/index.ts
相关: Telemetry

消息和类型术语

Message 概念
消息是对话的基本单元。包括用户消息、助手消息、系统消息、工具调用消息等多种类型。
types/message.ts: Message 类型定义
相关: User Message, Assistant Message
User Message 概念
用户消息，包含用户输入的文本、图片、文件等内容。
{ type: 'user', content: '...' }
相关: Message
Assistant Message 概念
助手消息，包含 AI 模型的响应，可能包括文本、工具调用等。
{ type: 'assistant', content: [...] }
相关: Message
System Message 概念
系统消息，包含系统提示、上下文信息等。通常用于设置 AI 的行为和能力。
{ type: 'system', content: '...' }
相关: System Prompt
Context Window 概念
上下文窗口，AI 模型一次能处理的最大 token 数量。Claude Code 需要管理这个限制。
utils/context.ts: getContextWindowForModel()
相关: Compact, Token

UI 和交互术语

Ink UI
React for CLI，一个用 React 组件构建命令行界面的框架。Claude Code 使用 Ink 渲染终端 UI。
ink/ 目录
相关: Component
Component UI
React 组件，用于构建 UI 的可复用单元。Claude Code 有大量组件用于终端渲染。
components/ 目录
相关: Ink
Diff UI
差异显示，用于展示文件修改前后的变化。Claude Code 有专门的 Diff 组件用于代码审查。
components/diff/ 目录
相关: File Edit
Permission Dialog UI
权限对话框，用于在执行敏感操作前请求用户确认。
components/permissions/ 目录
相关: Tool Permission

高级概念

Skill 概念
技能，一种可复用的能力模块。技能可以包含提示、工具、工作流等，用于扩展 Claude 的能力。
skills/ 目录
相关: Plugin
Task 概念
任务，用于管理后台执行的工作单元。支持创建、查询、停止等操作。
tasks/ 目录, TaskCreateTool
相关: Agent Tool
Subagent 概念
子代理，由 Agent Tool 创建的独立 AI 实例。子代理有自己的上下文和执行环境。
tools/AgentTool/ 目录
相关: Agent Tool, Task
Hook 概念
钩子，允许在特定事件发生时执行自定义逻辑。Claude Code 支持多种钩子类型。
hooks/ 目录
相关: Plugin
Telemetry 概念
遥测，收集使用数据用于分析和改进。包括错误报告、性能指标、功能使用情况等。
utils/telemetry/ 目录
相关: Analytics
